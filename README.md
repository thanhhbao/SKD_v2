# My Training Repository

This repository is a modified version for training purposes. It contains customized code for learning and experimentation. **Please do not clone this repository directly.** If you wish to use the original project, visit the main repository: [SKD Repository](https://github.com/thanhhbao/SKD).

## About

The SKD framework provides a modular, configurable pipeline for skin cancer detection and other medical image classification tasks, with a focus on efficient metrics logging and experiment tracking. This repository includes my custom modifications for educational purposes.
